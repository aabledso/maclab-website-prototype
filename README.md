# MacLab Website Prototype

## Purpose
MacLab is a fictional fast food restaurant concept focused on macro-friendly and healthy meals. This project is a two-page website prototype built using HTML, CSS, and Markdown.  

This project aligns with the defined scope and deliverables outlined in the documentation folder.

## Website Structure
The website includes:
* 'index.html' - Homepage for MacLab
* 'about.html' - About page including menu items
* 'style.css' - Stylesheet used for both pages  

All files organized according the project requirements.

## Project Documentation
All project documentation can be found in the 'docs/' folder:
* 'scope.md' - Final project scope statement
* 'plan.md' - Risk analysis and task schedule
* 'restrospective.md' - Reflection on project process  

The documentation demonstrates structured planning and risk awareness.
